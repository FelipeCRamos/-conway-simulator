Gen* parse ( std::ifstream &  config_file)

It's a parse function, responsible for reading the initial config file passed through the execution parameter (**argv). It detects the Height and Width of the given canvas and then detects what is the char that will represent the alive cells. Anything different than the char, will be read as a dead cell.
